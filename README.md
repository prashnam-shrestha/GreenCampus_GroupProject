# Green Campus Helper

A simple console-based Java application to encourage and track eco-friendly activities on campus.  
Built as a group project for the **Object Oriented Programming** course (ITS63304 / ITS69704) at Taylor's University.

## Features

- User registration and login (Student / Admin roles)
- Log daily green activities (Walking/Cycling, Saving Electricity, Using Reusable Materials)
- Earn points for each eco-friendly action
- View personal activity history and total points (for students)
- Admin dashboard: view all activities and generate campus sustainability reports
- Most popular eco-action statistics

Demonstrates core OOP concepts:
- Encapsulation
- Inheritance & Polymorphism
- Abstract classes
- Interfaces

## Technologies

- Java (JDK 17+ recommended)
- Maven (for build & dependency management)
- Console-based UI (no GUI)
